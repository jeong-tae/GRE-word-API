# GRE-word-SLACK API
This code is based on python3.x
### pre-requisites
> pip install slacker
> mkdir data
You should add data files into this "data" directory.

### Compile
Then, 
> python send_quiz.py "{word|meaning|cloze|synonyms}"

Or you can send the quiz set daily as adding a command on crontab.
